# Система оценки рисков

Веб-приложение для оценки рисков активов с использованием экспертных оценок и многокритериального анализа.

## 🚀 Быстрый старт

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python app.py
```

Откройте браузер: `http://localhost:5000`

**Учетные данные по умолчанию:**
- Администратор: `admin` / `admin123`

## 📋 Основные возможности

- ✅ Управление активами и экспертами
- ✅ Экспертные оценки по 6 критериям
- ✅ Оценка вероятностей угроз
- ✅ Расчет критичности и рисков
- ✅ Визуализация в виде тепловых карт
- ✅ Система аутентификации с ролями
- ✅ Разграничение доступа (админ/эксперт)
