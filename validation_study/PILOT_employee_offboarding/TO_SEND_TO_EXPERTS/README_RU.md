# Что отправлять экспертам

Эта папка содержит только файлы, которые можно показывать экспертам. В ней нет
скрытых файлов с расчётом системы, ground truth или итоговыми категориями.

## Сначала, до Task 1/2

Отправьте каждому эксперту:

- `EXPERT_INSTRUCTIONS_RU.md`
- `00_process_description.md`

Затем выберите один формат сбора ответов:

- бумага: `EXPERT_PACKET_print_this.md`;
- веб-форма: `expert_vote_task1_task2_RU.html`.

Эксперты должны работать независимо и не обсуждать ответы друг с другом.

## После Task 1/2

Только после того, как все эксперты завершили Task 1 и Task 2, покажите им
расчёт системы по этому процессу. Затем отправьте/выдайте:

- бумага: `task3_sus_questionnaire.md`;
- веб-форма: `web_form_task3_sus.html`.

## Не добавляйте сюда

Не отправляйте экспертам файлы из родительской папки:

- `ground_truth_scoring_rationale.md`
- `framework_categories.csv`
- `task1_framework_scoring_TOFILL.csv`
- `task2_holistic_rating_TOFILL.csv`
- `task3_sus_TOFILL.csv`
- `ADMIN_INSTRUCTIONS_RU.md`
