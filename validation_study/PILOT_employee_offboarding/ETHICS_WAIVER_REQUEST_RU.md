# Заявка на этическое заключение / waiver — черновик письма

Отправьте это (или адаптированный вариант) в этический комитет вашего университета
(обычно — при управлении науки/докторантуры или при профильном институте). Цель —
получить официальное заключение "exempt" / "не требует полного рассмотрения" —
для этого почти всегда достаточно короткого запроса, а не полной заявки, если
исследование действительно минимального риска (как в данном случае).

---

**Тема письма:** Запрос на этическое заключение (exempt review) для пилотного
исследования в рамках статьи по риск-менеджменту

Уважаемые коллеги,

Прошу рассмотреть возможность выдачи заключения об отсутствии необходимости
полного этического рассмотрения (exempt determination) для пилотного
исследования, проведённого в рамках подготовки статьи для международного
рецензируемого журнала (Applied Sciences, MDPI).

**Краткое описание исследования.** Пилотное исследование сопровождает разработку
программной системы поддержки принятия решений для оценки рисков бизнес-процессов
(грант Комитета науки МНВО РК № BR24992852). В исследовании приняли участие 8
профессионалов (ИТ-инфраструктура, информационная безопасность, HR, комплаенс) с
опытом работы от 8 до 15 лет. Участники оценивали гипотетический, стандартизированный
сценарий бизнес-процесса ("увольнение сотрудника и отзыв доступа") — не реальные
данные какой-либо организации — по формализованной шкале и в свободной форме, а
также заполнили стандартизированную анкету юзабилити (System Usability Scale).

**Почему это минимальный риск.**
- Собирались только профессиональные суждения участников о гипотетическом кейсе —
  не персональные данные, не данные о здоровье, не сведения об уязвимых группах.
- Участие анонимно: ответы фиксировались под условным ID (P1–P8), без фамилий.
- Ни организация, ни работодатель, ни личность участников не раскрываются и не
  восстановимы из опубликованных данных.
- Участие добровольное; участники были заранее проинформированы о цели
  исследования, анонимности и агрегированном характере публикации результатов.

**Что нужно.** Официальное письмо/номер заключения об exempt-статусе (или,
если требуется, ускоренное рассмотрение) для указания в разделе "Institutional
Review Board Statement" рукописи при подаче в журнал.

Готов(а) предоставить полный протокол исследования, анкеты и инструкции для
участников (все материалы уже подготовлены и приложены/доступны по запросу).

С уважением,
[ФИО, должность, кафедра/институт]
