# Система оценки рисков

Веб-приложение для оценки рисков активов с использованием экспертных оценок и многокритериального анализа.

## 📋 Описание

Система позволяет:
- Управлять активами и экспертами
- Собирать экспертные оценки активов по 6 критериям
- Оценивать вероятности угроз
- Рассчитывать критичность и риски активов
- Визуализировать результаты в виде тепловых карт
- Управлять мерами контроля и анализом рисков

## 🚀 Быстрый старт

### Требования

- Python 3.7+
- pip

### Установка

1. Клонируйте репозиторий или распакуйте проект

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Инициализируйте базу данных (выполняется автоматически при первом запуске)

4. Заполните справочники (опционально):
```bash
python add_threats.py
python populate_vulnerabilities.py
python populate_control_measures.py
```

5. Запустите приложение:
```bash
python app.py
```

6. Откройте браузер и перейдите по адресу: `http://localhost:5000`

## 👥 Роли пользователей

### Администратор
- Полный доступ ко всем функциям системы
- Управление пользователями, экспертами и активами
- Просмотр всех оценок и результатов
- Управление анализом рисков

### Эксперт
- Просмотр и редактирование только своих оценок
- Оценка активов по 6 критериям (0-10)
- Оценка вероятностей угроз (1-3)
- Просмотр результатов критичности и рисков

## 📊 Основные функции

### 1. Управление активами
- Добавление, редактирование и удаление активов
- Автоматический расчет средних оценок на основе экспертных оценок

### 2. Экспертные оценки
- Оценка активов по критериям:
  - Жизнь/Здоровье
  - Экономика
  - Экология
  - Зависимость
  - Социальное
  - Международное
- Оценка вероятностей угроз (1-3)

### 3. Расчет рисков
- Критичность = взвешенная сумма критериев
- Impact = 1 + (критичность / 10) × 2
- Риск = Impact × Вероятность
- Остаточный риск = Риск × (1 - эффективность_контроля)

### 4. Классификация рисков
- **Низкий** (1.0-3.9): Допустимый, контрольный
- **Средний** (4.0-6.9): Требует мер по снижению
- **Высокий** (7.0-9.0): Требует приоритетного устранения

### 5. Визуализация
- Тепловые карты остаточных рисков
- Цветовая индикация уровней риска

## 🗄️ Структура базы данных

Основные таблицы:
- `users` - пользователи системы
- `experts` - эксперты
- `assets` - активы
- `asset_evaluations` - оценки активов экспертами
- `threat_probabilities` - вероятности угроз
- `threats` - справочник угроз
- `vulnerabilities` - справочник уязвимостей
- `control_measures` - меры контроля (ISO 27001)
- `risk_analysis` - анализ рисков
