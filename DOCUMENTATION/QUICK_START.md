# Быстрый старт

## 🚀 За 5 минут

### 1. Установка
```bash
pip install -r requirements.txt
```

### 2. Запуск
```bash
python app.py
```

### 3. Вход
- URL: `http://localhost:5000`
- Логин: `admin`
- Пароль: `admin123`

## 📋 Основные действия

### Для администратора

1. **Создать эксперта**
   - Эксперты → Добавить эксперта

2. **Создать пользователя для эксперта**
   - Пользователи → Добавить пользователя
   - Роль: Эксперт
   - Выбрать эксперта из списка

3. **Добавить актив**
   - Активы → Добавить актив

4. **Просмотреть результаты**
   - Критичность и риски

### Для эксперта

1. **Оценить актив**
   - Оценки активов → Добавить оценку
   - Заполнить все критерии (0-10)

2. **Оценить вероятность угрозы**
   - Вероятности угроз → Добавить вероятность
   - Значение: 1-3

## 🔢 Формулы

- **Критичность** = Σ(оценка × вес)
- **Impact** = 1 + (критичность / 10) × 2
- **Риск** = Impact × Вероятность
- **Остаточный риск** = Риск × (1 - эффективность)

## ⚠️ Важно

- Эксперты видят только свои оценки
- Каждый эксперт может оценить актив только один раз
- Администратор видит все данные
- Пароль администратора нужно изменить в продакшне
